# Security Policy

AIWiki 是一个本地 CLI 工具。它会按用户指定的路径写入 Markdown、JSON 和配置文件。

## 安全边界

AIWiki 不会：

- 主动上传用户本地知识库内容
- 绕过登录墙或付费墙
- 自动修改 `.obsidian` 插件配置
- 自动安装 Dataview
- 扫描未指定的个人目录

## 建议

- 只在你信任的环境里运行 AIWiki
- 首次使用时最好指定一个新的测试知识库目录
- 遇到异常写入行为时，先停下来检查 payload 和目标路径

## 报告问题

如果你发现疑似安全问题，请优先提交 issue，并尽量附上：

- 运行命令
- 目标路径
- `aiwiki doctor` 输出
- `aiwiki status` 输出
- 能稳定复现的最小示例

如果问题涉及本地数据泄露、越权写入或路径逃逸，请尽量把细节说清楚，方便快速复核。
